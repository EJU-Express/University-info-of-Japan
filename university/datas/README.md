# University-info-of-Japan

University info of Japan [uij-data.eju-express.com](https://uij-data.eju-express.com/). Data from mext.go.jp.

# 説明

こちらは整理したJSONデータです。

Mongodbに相性がいいと思います。

大学の種類にも拘わらず、一つファイルにまとめました。
そして、大学の**英語名も半角のスペース**に統一されました。

**フォーマットしたデータは10万行以上になりますのでご注意ください。**

[./r4-university-info.json](r4-university-info.json)

# xx-convertor.py

転換用スクリプトはこちらにあります。
ご参考にどうぞ。

ソースデータは年度によって少し形もかわりますので、別々に作りました。

* [r4-convertor.py](r4-convertor.py)
